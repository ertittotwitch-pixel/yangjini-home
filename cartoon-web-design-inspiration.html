⚙️ 卡通站常用技术栈

🎬 Lottie / dotLottie

矢量动画，文件仅为 GIF 的 1/10-1/30。支持状态机，idle/wave/celebrate 一键切换。跨平台。

LottieFiles 指南 →

🕹️ Rive

实时交互动画工具，状态控制和输入响应比 Lottie 更灵活。适合复杂角色动画。

rive.app →

📜 GSAP + ScrollTrigger

滚动驱动卡通叙事的主力工具。逐帧动画 + 视差 + 时间线控制。

gsap.com →

🧊 WebGL (Three.js / OGL)

3D 场景 + 手绘纹理贴图。案例：ITom 作品集、Ponpon Mania 梦境场景。

threejs.org →

✍️ SVG 动画 (CSS / SMIL)

轻量手绘线条动画，stroke-dasharray + scroll-timeline 实现零 JS 依赖。

🖌️ Canvas 2D

像素艺术、涂鸦效果、手写笔迹。适合低性能开销的满屏卡通特效。

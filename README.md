# 羊积泥 · 个人主页

靠跟 AI 聊天写代码的大学生做的个人介绍页。vibe coding 主题，手工质感 + 非对称布局。

## 技术栈

纯 HTML + CSS + JS，零依赖。

## 设计特点

- 纸纹背景（SVG feTurbulence）
- 手写字体标注（Ma Shan Zheng）
- 六个 section：关于 / 来路 / 涉猎范围 / 折腾过的东西 / 当下 / 我认为
- IntersectionObserver 滚动渐现
- Hero 鼠标视差
- 响应式适配

## 版本

- `yangjini-home.html` — v1，温和人格版
- `yangjini-home-v2.html` — v2，加入完整内容 + 动效
- `yangjini-home-v3.html` — v3 当前版本

## 本地预览

直接用浏览器打开任意版本的 HTML 文件即可。
